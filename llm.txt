# KTS Cleaning Website (ktscleaning.be)

Professional cleaning services website built with Astro, focusing on specialized cleaning services including crime scene cleaning, industrial cleaning, and specialized residential cleaning in Belgium. **Performance optimized to achieve 95+ Lighthouse scores**.

### Cache Strategy
- **Immutable cache** for static assets (31536000s)
- **Font files** cached for 1 year with immutable flag

- **Video captions** and accessibility attributes

## Content Management
- **Blog content**: Markdown files in `/src/content/blog/`
- **Client data**: JSON file at `/src/data/client.json`
- **Navigation data**: JSON file at `/src/data/navData.json`
- **CMS**: Netlify CMS for content editing
- **Content validation**: Zod schemas for type safety

## Styling & Assets
- **CSS Framework**: LESS preprocessor with modular stylesheets
- **Fonts**: Lato font family (300, 400, 700, 900 weights) with async loading
- **Colors**: 
  - Primary: #fed500 (yellow)
  - Secondary: #2a2400 (dark contrast-friendly)
  - Header: #1a1a1a (dark text)
  - Body: #4e4b66 (readable gray)
- **Images**: Optimized WebP/AVIF formats, responsive sizing
- **Animations**: Modern Intersection Observer API instead of heavy libraries
